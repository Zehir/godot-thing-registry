# godot-thing-registry
A modular editor plugin to define, extend, and manage gameplay “things” such as items, skills, buffs, states, and inventories.


## Godot version
This addon will target a futur version of Godot, I think 4.7 because I am currently using a custom builds. You can find the version I am using at https://github.com/Zehir/godot/tree/dev
